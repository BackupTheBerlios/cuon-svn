Kurze Einführung in Virtalisierer

Ein Virtualisierer ist ein Programm, das einen PC simuliert. Man kann also z.B. auf Linux das Programm VirtualBox starten und dann dort Microsoft Windows installieren, wenn man z.B. bestimmte Windows-Programme benötigt. Aber natürlich auch andersherum, man installiert auf Windows z.B. dem VM-Player und startet unser CUON-Image und damit eine komplette LINUX-Umgebung. Die Virtualisierer bieten auch Netzwerkzugriff, so das es wie ein eigenständiger computer wirkt, mit dem man ins Internet gehen kann, der aber auch von außen, z.B. dem Firmennetzwerk erreichbar ist.
Sie können also auf Ihren Computer, sei es nun ein Linux- oder Windows-PC einfach das CUON-Image starten und ganz normal arbeiten.
