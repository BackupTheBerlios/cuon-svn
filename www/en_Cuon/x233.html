Example 2-9. Create Grants File

Services --> Database Tools
There Edit configfiles --> Server-->SQL-Rights-->Create Grant File

and write the new database rights.

after this you should do connect to the database and then choose: Tools-->Database-->Database-->SQL-Server-->grants

Look also at "User create and attend".

Set here some rights for the DMS, look also at the "sql.ini"
