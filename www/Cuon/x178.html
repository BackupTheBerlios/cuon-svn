1.10. Update

Benutzen sie zum Update unbedingt den user und Rechner auf dem das Cuon Setup Konfigurationstool installiert ist. Starten Sie das Cuon Setup tool aus dem Menü 'Einstellungen'. Oder vom Terminal aus: Dazu in das Verzeichnis cuon_client wechseln und mit "python setup.py" die graphische Installationsroutine starten.

Figure 1-1. Setup tool

1.10.1. Eigentliches Update

"Start Install" drücken. Genauso wie bei der Installation, muss dazu ein funktionierender SSH Zugang eingerichtet sein!! Es sollte also auf jeden Fall der o.g. Cuon-Admin-Bereich sein.

Hier im Beispiel ist cuon als SSL (Protocoll https) Server installiert. Diese Version ist umbedingt zu empfehlen, wenn Clients aus dem Internet auf den Server zugreifen sollen.

1.10.2. Client "bauen"

Services -> Create Client

1.10.3. Datenbank vorbereiten

Wichtig an diesem Schritt:

  • mittels "Connect" mit Admin User (!!) und Passwort anmelden.

  • unter "Edit Configfiles -> Server" die [:Konfigurationsdateien:Konfigurationsdateien ]anpassen.

Danach weiter zum nächsten Schritt: Tools -> Database

1.10.4. Datenbank

Figure 1-2. Setup Tool Database

Bei einer Neuinstallation oder wenn Userdaten geändert wurden, neue User hinzugefügt oder ähnliches, dann bitte auf jeden Fall im Bereich SQL-Rights den Punkt create Grants file ausführen.

Dann nur noch "SQL-Server -> start complete update" ausführen.

Fertig!
