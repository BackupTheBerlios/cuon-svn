Neues rund um C.U.O.N.

01.04.2011

Bei den Artikel Stammdaten können jetzt auch größere Textpassagen als Notiz hinterlegt werden.

29.03.2011

In den reports sind nun kurze Python Befehle möglich, Leerzeichen dürfen nicht enthalten sein. Für längere Python Befehle bitte statt der class "Function" die class "Python" nehmen.

13.03.2011

Das Firmenlogo kann nun im DMS mit dem Titel "cuon_mainwindow_logo" angelegt werden, es kann ein übliches format haben, jpg oder png beispielsweise.

11.03.2011

Mit den neuen Variablen in der "server.ini" kann jetzt gezielt festgelegt werden, on http genutzt wird oder nur das sichere https.
[XMLRPC]
ALLOW_HTTP: NO
HTTP_PORT:7080
HTTPS_PORT:7580
HOST:localhost

Ebenfalls können die Ports für http und https festgelegt werden.

02.03.2011

Im Report-Gnerator können jetzt unterschiedliche Hintergrundbilder festgelegt werden und jeweils über das Property Tag angesteuert werden.

20.02.2011

Um schneller den Zusammenhang zwischen den dokumenten im DMS und dem cuon Modul zu nutzen, gibt es jetzt den Button "Jump to Origin", mit dem öffnet sich das entsprechende Modul, z.Z. Adressen, Aufträge und Projekte. To be continued ....

16.02.2011

Bei den Adressen kann jetzt eine Preisgruppe hinterlegt werden, ebenfall bei den Gräbern und noch diese woche bei den Aufträgen. Damit ist es möglich, je nach Auftrag oder Adresse unterschiedliche Preise zu berechnen.

14.02.2011

Es gibt jetzt eine Exportfunktion, die Daten aus cuon heraus in bestimmte Formate exportiert, gestuert wird dies über eine Exportdatei.

26.01.2011

Das Mail2DMS unterstützt nun auch html E-Mails,damit wird jetzt ein weiter Bereich an E-Mails erfaßt.Für Admins gilt, das zusätzlich html2text auf dem Server installiert sein muß.

25.01.2011

In der Adressverwaltung können nun mehrere Email-Adressen für die Adresse und jeweils den Gesprächspartnern hinterlegt werden. Das DMS sortiert dann emails auch nach diesen Adressen und sichert die Dokumente.

11.01.2011

Es steht nun ein komplettes Cuon System als VirtualBox Image im Download Menüzur Verfügung. Das Image ist gepackt , aber dennoch über 2 GB groß.

11.01.2011

Das DMS bildet nun automatisch einen Extract von neu angelegten PDF, txt und odt Dokumenten.

10.01.2011

Fehler im xgettext Modul mit Bindung an libexpat beseitigt, Übersetzungen gehen nun wieder.

08.01.2011

Version 11.01.09 wurde ins SVN übertragen. Das aktualisieren wird vom Administrator über das cuon-setup Programm ausgef.

08.01.2011

In der Benutzerverwaltung kann der Splashscreen mit den cuon News abgeschaltet werden.

06.01.2011

Bei der Projektverwaltung wurde in die Übersicht der Projekt-Status eingebunden. Ebenfalls kann man nun in allen Listenansichten die Header (Überschriften) anklicken und so die Sortierreihenfolge ändern. So kann jetzt z.B. in der Adressverwaltung nach Straßennamen sortiert werden.

05.01.2011

Der eingebaute Editor ( für Notizen, Konfigurationsdateien, Reportvorlagen ) ist stark erweitert worden. Neben XML Highlightning ist ebenfalls ein XML Beautifier vorhanden. Ebenfalls wurde ein auto-indent eingebaut. Weiterhin wird nun neben python auch C, C++,Perl, SQL usw. unterstützt und farblich hervorgehoben.

30.12.2010

Die automatische Übername von emails mit Anhängen wurde verbessert, nun können getrennte Werte für den Imap-Server und dem Post-Ausgangsserver gesetzt werden.

24.12.2010

Von jetzt an sind auch automatische Logins möglich, der Config-Wizard hat nun auch Felder für den Benutzernamen, das Passwort und die Mandanten ID
