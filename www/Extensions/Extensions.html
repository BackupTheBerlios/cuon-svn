Erweiterungen

Mit C.U.O.N. steht Ihnen ein umfangreiches kaufmännisches System zur Verfügung, das Sie mit allen XMLRPC-fähigen Programmiersprachen erweitern können.Hier eine kleine Auswahl:

Python, Ruby,Java, C, C++, Perl, Delphi, .NET,PHP, LUA, TCL

C.U.O.N. stellt Ihnen dabei den kompletten Unterbau bereit, insbesondere fertige Module für die Adressenpflege, Termine, Personal, Dokument Management System, Artikel, Literaturverwaltung, Rechnungswesen und Lager. Dazu kommen dann noch die Server, angefangen mit dem XMLRPC-Server bis zum Reportserver.
Wenn Sie also ein Modul zu C.U.O.N. hinzufügen möchten, nimmt Ihnen C.U.O.N. einen Großteil der Routinearbeit ab:
  1. Datenbank anlegen, Tabellen anlegen und pflegen
  2. Benutzerverwaltung, Zugriffsrechte
  3. Serverroutinen zum Daten einfügen, ändern, löschen
  4. Viele Hundert andere, nützliche Serverroutinen
  5. Direkter Zugriff mit SQL-Befehlen über den Server
  6. Zugriff auf alle Datentabellen
  7. Das Grundsystem besitzt eine Softwareverteilung
  8. Internetfähig
  9. Fernwartung
Also können sie sich auf das wesentliche, nämlich Ihr eigenes Modul, konzentrieren.
Sie sehen also, das Sie mit C.U.O.N. innerhalb kürzester Zeit Ihr Wunschprogramm entwickeln können oder eine passende Erweiterung vornehmen können.
