5.3. Umgebungsvariablen

Setzen der Umgebungsvariable (Beispiele) :

1. Linux:

in der .bashrc setzen sie bitte die Variablen (Hier mit Beispiel)

nur Server "export CUON_INI=/etc/cuon/cuon.ini "

Client und Server "export CUON_HOME=~/cuon "

Client und Server "export CUON_SERVER=192.168.17.251:9673/Cuon, Hier gilt also IP_Adresse:Port/Verzeichnis

nur Client CUON_OOEXEC=/opt/OpenOffice.org1.1.0/program/python, dies ist das eingebaute Python von Openoffice für die uno-bridge

Wichtig ist hierbei, das sich der Wert Server auf den Rechner bezieht, von dem aus das Program cuon_server gestartet wird. Dies kann durchaus ein Computer sein, der auch als client definiert ist oder eine normale Arbeitsstation ist. ( entsprechend muß die ip in cuon.ini angepaßt werden.
