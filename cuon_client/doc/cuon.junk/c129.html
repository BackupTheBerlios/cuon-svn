Kapitel 6. Benutzer

Zuerst muß ein Benutzer angelegt werden. Ab Version 0.24 geschieht dies über die Datei grants.xml, später ist eine seperate Datei vorgesehen. Editeren sie die XML-Datei mit einem beliebige Editor und legen sie Ihren Benutzer an. Ordnen sie ihn im zweifelsfalle der Gruppe cuon_all zu. Damit kann er so ziemlich alles in C.U.O.N. machen.
